## Definition
JavaScript (JS) is a **high-level**, **interpreted**, and **dynamic** programming language

It runs **both in browsers and on servers (Node.js)**.

## Why JavaScript?
- The **only language that runs natively in all browsers**.  
- Used for both **Frontend (React, Vue)** and **Backend (Node.js, Express)**.
